
# 🚦 E-Challan Analytics Dashboard
A full-stack data analytics and visualization dashboard that analyzes Indian traffic E-Challan data using Python, Pandas, Flask, and Bootstrap. The project transforms raw traffic violation data into meaningful insights such as trends, efficiency, revenue patterns, seasonality, and court case analysis.

---

## 📌 Project Overview

This system processes E-Challan datasets and generates interactive dashboards with key performance indicators (KPIs) and visual analytics. It helps understand traffic violation patterns, revenue generation, and enforcement efficiency over time.

---

## ✨ Key Features

- 📊 Time Series Analysis of Traffic Challans
- 💰 Revenue Trend Analysis
- ⚖️ Pending vs Disposed Case Tracking
- 📈 Efficiency Calculation (Disposal Rate)
- 📅 Seasonality Detection (Month-wise patterns)
- 🧠 Correlation Analysis between metrics
- 📉 Growth Rate Analysis (Monthly & Yearly)
- 📊 Interactive Web Dashboard using Flask
- 🎨 Beautiful UI with Bootstrap 5
- 🌐 Interactive Flask dashboard

⚙️ Installation & Setup
1. Clone Repository
git clone https://github.com/your-username/echallan-analytics-dashboard.git  
cd echallan-analytics-dashboard
2. Install Dependencies
pip install -r requirements.txt
3. Run Flask App
python app.py
4. Open in Browser
http://127.0.0.1:5000/
